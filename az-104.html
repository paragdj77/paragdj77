Course Overview

The AZ-104 Azure Administrator certification is designed for Azure administrators who manage cloud services spanning storage, security, networking, and compute cloud capabilities. This role involves implementing, managing, and monitoring identity, governance, storage, compute, and virtual networks in a cloud environment.

Prerequisites

  • • 6+ months of hands-on experience with Azure
  • • Understanding of core Azure services
  • • PowerShell, Azure CLI, and Azure Portal knowledge
  • • ARM templates and Azure Resource Manager
