Course Overview

The AZ-900 Azure Fundamentals certification is designed for individuals who want to demonstrate foundational knowledge of cloud concepts, core Azure services, security, privacy, compliance, and Azure pricing and support. This course is perfect for beginners with no prior Azure experience.

Who Should Take This Course?

  • • IT professionals new to Azure
  • • Sales and marketing professionals
  • • Students and recent graduates
  • • Anyone interested in cloud computing

  • Cloud Service Models (IaaS, PaaS, SaaS)
  • Cloud Deployment Models
  • Shared Responsibility Model
  • Serverless Computing

Module 2: Azure Architecture

  • Azure Regions and Availability Zones
  • Azure Resource Groups
  • Azure Subscriptions
  • Management Groups
  • Azure Resource Manager

Module 3: Core Azure Services

  • Azure Compute Services
  • Azure Storage Services
  • Azure Networking Services
  • Azure Database Services
  • Azure App Services

Module 4: Azure Security

  • Azure Security Center
  • Azure Sentinel
  • Azure Key Vault
  • Azure Information Protection
  • Network Security Groups

Module 5: Identity and Access

  • Azure Active Directory
  • Multi-Factor Authentication
  • Conditional Access
  • Role-Based Access Control
  • Identity Protection

Module 6: Governance and Compliance

  • Azure Policy
  • Azure Blueprints
  • Cloud Adoption Framework
  • Privacy and Compliance

Module 7: Pricing and Support

  • Azure Pricing Calculator

  • Azure Service Level Agreements
  • Cost Management Tools
