Course Overview

The AZ-700 Designing and Implementing Microsoft Azure Networking Solutions certification is designed for network engineers who specialize in designing, implementing, and maintaining Azure networking solutions. This role involves implementing core networking infrastructure, routing, Azure VPN Gateway, ExpressRoute, and network security.

Prerequisites

  • • Strong understanding of networking fundamentals
  • • Experience with Azure networking services
  • • Knowledge of on-premises networking technologies
  • • Familiarity with PowerShell and Azure CLI
  • • Understanding of security best practices

Module 4: ExpressRoute

Module 5: Network Security
